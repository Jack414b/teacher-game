# 教师编制闯关大挑战 — 设计规格书

**日期**: 2026-07-10 | **状态**: 已确认

## 1. 项目概述

一款像素 RPG 风格的纯前端 Web App，将教师编制备考过程游戏化。
玩家通过完成每日备考任务获取虚拟货币，兑换真实奖励。
Boss 可通过暗号进入后台查看数据、发放奖励。

## 2. 技术栈

| 层 | 选型 |
|---|------|
| 前端框架 | React 18 + TypeScript |
| 构建工具 | Vite 5 |
| UI 风格 | 像素 RPG 风（CSS 像素边框 + 自研组件） |
| 云数据库 | Supabase（PostgreSQL） |
| 状态管理 | Zustand |
| 路由 | React Router v6 |
| 部署 | Vercel |

## 3. 核心模块

1. 主页仪表盘 — 货币余额、今日进度
2. 每日打卡 — 6 项任务、日历热力图
3. 转盘抽奖 — Canvas 像素转盘动画
4. 兑换商城 — 小豆区/大豆区
5. Boss 后台 — 数据总览、手动发补贴
6. 设置 — 数据备份、规则说明
